Koa11y

Description:

Koa11y is a desktop application that is used to assess compliance with accessibility standards on a website

Contributions:

Enhanced features and added functionality specific to the Mac OS. Refactored the code base into modules to improve maintainability and readability. Implementing a new API engine(aXe) to perform the accessibility audits.

Technologies:

Vue, PhantomJS, Node, npm, GitHub, FS-Extra, aXe, pa11y

Spry Science

Description:

Spry Science is a web application that consumes data from the Clinical Trials government API. The data is stored and made searchable by customers who can log in to an account and perform customized searches.

Contributions:

Originally a 100% Ruby on Rails project, I rebuilt the front end in React, adding new sign in functionality and a sign up modal, integrating the React front end with the Rails server API. Installed HighCharts.js to render graphical data.

Technologies:

React, React-Bootstrap, Ruby on Rails, Postgres, HighCharts.js, Superagent, npm, GitHub, Heroku

Game of Life

Description:

Game of Life uses React in combination with Redux. It represents John Conway's model of the life and death of organisms.

Contributions:

Originally a React only project, I used this opportunity to learn Redux for all state management, which simplified the code and increased maintainability. The project is hosted on Heroku. Webpack-dev-server was used in the development process.

Technologies:

React, Redux, npm, Heroku, Webpack, Babel, ES6

Simon

Description:

This is a replica of the original Simon game. The styling and logic were all built from scratch.

Contributions:

This project is an example of using CSS to create the Simon image on the screen. There were also several challenges to implement functionality using Javascript and JQuery, such as an on/off button, resetting the game, and changing to a special "strict" mode, all triggered through click events.

Technologies:

HTML, CSS, JQuery, Javascript
